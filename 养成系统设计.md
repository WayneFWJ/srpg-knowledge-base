# 养成系统设计

> SRPG 角色养成系统深度设计指南
> 转职/装备/羁绊/长线运营设计

---

## 一、养成系统概述

### 1.1 养成系统定位

```
养成系统核心目标：
├─ 角色成长感：让玩家感受到变强
├─ 长期目标：提供持续的追求动力
├─ 策略深度：通过选择创造Build差异
├─ 付费点：合理的商业化空间
└─ 社交话题：玩家间交流讨论内容

养成系统原则：
├─ 可见的成长：每次养成都有明显提升
├─ 渐进式解锁：避免一次性全部开放
├─ 资源循环：养成消耗与产出平衡
├─ 可逆性：错误选择有机会修正
└─ 差异化：不同选择产生不同Build
```

### 1.2 养成维度矩阵

```
养成维度分类：

基础养成（必选）：
├─ 等级系统：基础属性成长
├─ 星级/稀有度：成长上限解锁
└─ 技能等级：技能效果提升

进阶养成（可选）：
├─ 职业/转职：角色定位切换
├─ 装备系统：属性与特效
├─ 兵营/兵种：战斗单位强化
└─ 羁绊/好感：关系加成

深度养成（长线）：
├─ 魂石/符文：随机属性组合
├─ 附魔/精炼：装备深度强化
├─ 大心/觉醒：终极突破
└─ 皮肤/外观：个性化展示
```

## 二、等级与成长系统

### 2.1 等级系统设计

**等级曲线设计**：
```
经验需求曲线类型：

线性增长：
Lv.1 → Lv.2: 100 exp
Lv.2 → Lv.3: 100 exp
...（适合简单游戏）

递增增长（推荐）：
Lv.1 → Lv.2: 100 exp
Lv.2 → Lv.3: 120 exp
Lv.3 → Lv.4: 150 exp
...（适合长线运营）

指数增长：
Lv.1 → Lv.2: 100 exp
Lv.2 → Lv.3: 200 exp
Lv.3 → Lv.4: 400 exp
...（适合有等级上限的游戏）
```

**属性成长公式**：
```
基础属性 = 初始值 + (等级-1) × 成长值 × 品质系数

示例：
初始攻击：100
每级成长：5
品质系数：SSR=1.2, SR=1.0, R=0.8

Lv.10 SSR角色攻击 = 100 + (9 × 5 × 1.2) = 154
Lv.10 SR角色攻击 = 100 + (9 × 5 × 1.0) = 145
```

### 2.2 星级/稀有度系统

**星级进阶设计**：
```
初始星级分布：
├─ SSR：初始4星，最高6星
├─ SR：初始3星，最高5星
└─ R：初始2星，最高4星

升星消耗（以SSR为例）：
4星→5星：需要角色碎片 × 50
5星→6星：需要角色碎片 × 100

碎片获取途径：
├─ 抽卡重复转化
├─ 活动奖励
├─ 商店兑换
└─ 特定关卡掉落
```

**稀有度与养成上限**：
| 稀有度 | 初始星级 | 最高星级 | 等级上限 | 技能等级上限 |
|--------|----------|----------|----------|--------------|
| N | 1星 | 3星 | Lv.30 | Lv.3 |
| R | 2星 | 4星 | Lv.50 | Lv.5 |
| SR | 3星 | 5星 | Lv.70 | Lv.8 |
| SSR | 4星 | 6星 | Lv.90 | Lv.10 |

## 三、转职系统设计

### 3.1 转职系统架构

**转职树设计（梦幻模拟战）**：
```
典型转职树示例：

          战士 (初始)
         /    |    \
      剑士   枪兵   骑兵
      / \     |     / \
   英雄 剑圣 骑士 龙骑
    |    |    |    |
   ...  ...  ...  ...

设计特点：
├─ 每个角色6+转职方向
├─ 转职不可逆但可随时切换
├─ 跨系转职保留部分技能
└─ 终极职业有独特机制
```

**五内系统（天地劫）**：
```
五内 = 天赋树 + 职业分支 + 技能池

神(烈/魂/迅/魔)
├─ 主内：解锁核心技能、提升兵刃上限
├─ 辅内1：属性成长 + 分支技能
└─ 辅内2：属性成长 + 分支技能

设计亮点：
1. 选择成本：60级前无法点满两路，必须做抉择
2. 重置机制：灵蕴石可重置，但资源有限，增加决策重量
3. 技能组合：不同五内路线 = 完全不同的角色定位

案例 - 冰璃：
上路【魂】：追击、气血、会心 —— 输出路线
下路【迅】：定魂(免死)、先攻护卫 —— 生存路线

PVE主流：主点下路拿【定魂】，副点上路
PVP方向：需要【灭剑血胧】则走上路
```

### 3.2 技能继承机制

```
技能类型：
├─ 职业技能：当前职业专属
├─ 通用技能：所有职业可用
└─ 继承技能：转职前职业保留

策略意义：
├─ 先转A职业拿强力技能
├─ 再转B职业保留A技能
├─ 构建独特技能组合
└─ 但需要消耗稀有资源（魔导石）

设计平衡：
├─ 限制可继承技能数量
├─ 继承技能效果打折
└─ 高阶技能无法继承
```

## 四、装备系统设计

### 4.1 装备架构

**装备部位设计**：
```
标准装备位（6件）：
├─ 武器：攻击/法攻
├─ 头盔：生命/物防
├─ 护甲：生命/物防
├─ 手套：攻击/命中
├─ 鞋子：移动力/闪避
└─ 饰品：特殊属性/特效

扩展装备位：
├─ 时装：外观（不影响数值）
├─ 称号：属性加成+展示
└─ 坐骑：移动力/属性
```

**装备品质与成长**：
```
品质等级：
├─ 白色（普通）
├─ 绿色（优秀）
├─ 蓝色（稀有）
├─ 紫色（史诗）
├─ 橙色（传说）
└─ 红色（神话）

强化系统：
├─ +1~+10：金币强化，100%成功
├─ +11~+15：需要材料，概率成功
└─ +16~+20：高级材料，低概率，失败不掉级

精炼/附魔：
├─ 主属性：固定提升
├─ 副属性：随机词条
└─ 特效：特殊效果触发
```

### 4.2 魂石/符文系统（天地劫）

```
魂石类型：
├─ 妖术师：几率减CD —— 适合高频施法
├─ 朱焰魔火：增伤 —— 纯输出
├─ 百眼翼魔：施加有害状态 —— 控制流
├─ 罗鬼：增伤+吸血 —— 续航
└─ ...（多种魂石类型）

设计意图：
├─ 同一角色不同魂石 = 不同定位
├─ 增加养成深度
└─ 策略搭配空间

随机属性：
├─ 每个魂石有随机副属性
├─ 可洗练重置属性
└─ 追求完美词条是长期目标
```

## 五、羁绊/好感系统

### 5.1 羁绊系统架构

**羁绊类型**：
```
角色羁绊：
├─ 特定角色同时上场有加成
├─ 羁绊等级提升加成效果
└─ 解锁特殊对话/剧情

阵营羁绊：
├─ 同阵营角色数量达标有加成
├─ 例如：上阵3个蜀国武将+10%攻击
└─ 鼓励阵营编队

关系羁绊：
├─ 历史关系还原（刘关张桃园结义）
├─ 特定角色组合触发特殊效果
└─ 增强剧情代入感
```

**好感度系统**：
```
好感度等级：
├─ 陌生 → 友好 → 亲密 → 信赖 → 至交

提升方式：
├─ 共同出战
├─ 赠送礼物
├─ 羁绊剧情
└─ 特殊事件

奖励内容：
├─ 属性加成
├─ 专属剧情
├─ 特殊语音
└─ 解锁技能
```

### 5.2 梦战羁绊系统

```
羁绊层级：
├─ Lv.1：无特效
├─ Lv.2：生命+5%
├─ Lv.3：攻击+5%
├─ Lv.4：解锁特殊技能
└─ Lv.5：全属性+10%

羁绊触发：
├─ 亲密度达标
├─ 特定角色在场
└─ 完成羁绊任务

战略意义：
├─ 鼓励多角色培养
├─ 增加阵容搭配深度
└─ 提供长期养成目标
```

## 六、兵营/兵种系统

### 6.1 兵营系统（梦幻模拟战）

```
兵营架构：
├─ 步兵营
├─ 骑兵营
├─ 枪兵营
├─ 飞兵营
├─ 弓兵营
├─ 刺客营
├─ 僧侣营
└─ 魔物营

养成维度：
├─ 兵种等级：基础属性
├─ 兵种科技：特殊效果
└─ 兵种装备：额外属性

兵种解锁：
├─ 初始兵种：低级兵种
├─ 高级兵种：兵营等级解锁
└─ 特殊兵种：活动/限定获取

策略意义：
├─ 英雄携带不同兵种 = 不同玩法
├─ 兵营养成影响所有使用该兵种的英雄
└─ 长期养成投资
```

### 6.2 兵种进阶系统

```
进阶条件：
├─ 武将达到一定等级
├─ 消耗同品质同兵种武将研习
└─ 特定材料

代表高级兵种：
├─ 虎豹骑：前两回合伤害+20%
├─ 西凉铁骑：剩余移动力转化为免伤
├─ 无当飞军：弓兵特殊，附带中毒
├─ 大戟士：枪兵特殊，反击伤害提升
└─ 白毦兵：盾兵特殊，受击恢复兵力

设计意义：
├─ 高级兵种改变武将定位
├─ 同武将有不同玩法路线
└─ 高级兵种成为养成目标
```

## 七、长线运营设计

### 7.1 内容更新节奏

```
月度更新：
├─ 新角色（2-4个）
├─ 新关卡/活动
└─ 系统优化

季度更新：
├─ 大型资料片
├─ 新玩法模式
└─ 平衡性调整

年度更新：
├─ 新资料片剧情
├─ 新系统
└─ 联动活动
```

### 7.2 角色获取与养成

**角色获取途径**：
```
抽卡（主要方式）：
├─ 新手池
├─ 常规池
├─ 限定池
└─ 活动池

其他途径：
├─ 活动赠送
├─ 碎片合成
├─ 商店兑换
└─ 成就奖励

保底机制：
├─ 小保底：90抽必出SSR
├─ 大保底：180抽必出当期UP
└─ 硬保底：300抽可兑换指定角色
```

**养成维度矩阵**：
```
角色完整养成：
├─ 等级（经验）
├─ 星级（碎片）
├─ 职业（转职材料）
├─ 兵营（兵种等级）
├─ 装备（强化/精炼）
├─ 附魔（随机属性）
├─ 羁绊（好感度）
└─ 大心（后期系统）

养成周期：
├─ 轻度玩家：3-6个月达到可用水平
├─ 中度玩家：1-3个月达到主力水平
└─ 重度玩家：2-4周达到毕业水平
```

### 7.3 赛季制设计

```
赛季周期：
├─ 每赛季约3个月
├─ 赛季末重置部分进度
└─ 新赛季新内容

赛季内容：
├─ 新武将：保持新鲜感
├─ 新战法：新Build可能
├─ 新兵种：战术变化
├─ 新事件：历史事件
└─ 平衡调整：环境轮换

保值设计：
├─ 核心武将长期有效
├─ 战法可传承复用
└─ 兵种解锁永久保留
```

## 八、养成与关卡匹配

### 8.1 养成验证设计

```
关卡难度与养成关系：
├─ 推荐战力 = 基准战力 × 难度系数
├─ 难度系数：普通1.0，困难1.2，噩梦1.5

验证层次：
├─ 数值验证：战力达标可过
├─ 策略验证：需要正确Build和策略
└─ 操作验证：需要精确操作

越级挑战：
├─ 允许低于推荐战力挑战
├─ 策略可以弥补数值差距
└─ 越级越多，策略要求越高
```

### 8.2 资源产出规划

```
资源类型与获取：
├─ 经验：日常副本、关卡掉落
├─ 金币：日常副本、任务奖励
├─ 装备：装备本、关卡掉落
├─ 材料：活动、商店、掉落
└─ 碎片：抽卡、活动、兑换

产出节奏：
├─ 日常产出：保证基础养成
├─ 活动产出：加速特定养成
└─ 付费产出：提供快速提升途径
```

## 九、付费设计

### 9.1 付费点设计

```
主要付费点：
├─ 角色招募：核心付费
├─ 体力购买：加速养成
├─ 资源礼包：快速提升
├─ 月卡/通行证：稳定付费
└─ 皮肤/外观：不影响战力

白嫖友好设计：
├─ 核心角色可通过活动获取
├─ 低稀有度角色也有价值
├─ 资源可通过时间积累
└─ 赛季制缩小新老差距
```

### 9.2 付费平衡

```
付费与免费差距控制：
├─ 角色可获取：付费提前获得，免费后续获得
├─ 数值差距：付费有优势但非碾压
├─ 策略重要性：策略可弥补部分数值差距
└─ 时间换资源：免费玩家可通过时间追赶

付费体验：
├─ 提前体验新内容
├─ 更快养成速度
├─ 更多尝试机会
└─ 外观个性化
```

## 十、设计原则总结

### 10.1 养成系统设计原则

```
1. 可见成长原则
   - 每次养成都有明显提升
   - 成长曲线平滑但有惊喜点

2. 选择有意义原则
   - 不同选择产生不同Build
   - 没有绝对的最优解

3. 资源循环原则
   - 消耗与产出平衡
   - 避免资源溢出或匮乏

4. 长期目标原则
   - 提供持续的追求动力
   - 分阶段解锁新目标

5. 可逆性原则
   - 错误选择有机会修正
   - 重置成本合理
```

### 10.2 养成与战斗关系

```
养成即策略：
- 养成选择影响战斗策略
- 不同Build应对不同关卡
- 关卡设计考虑养成维度

策略即养成：
- 策略验证养成价值
- 策略成功产生养成动力
- 失败引导养成方向
```
