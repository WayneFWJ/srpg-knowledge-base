## 🎯 核心知识点概览

### SRPG 核心特征
- **网格/格子系统**：战斗发生在离散化的网格地图上
- **回合制战斗**：玩家与敌人轮流行动
- **角色培养**：等级、装备、技能等成长要素
- **策略决策**：位置、地形、克制关系影响战斗结果

### 关卡三维设计
```
敌人（Enemies）：谁来阻挡玩家？
环境（Environment）：在哪里战斗？
机制（Mechanics）：如何获胜？
```

### 品类标配机制
| 机制 | 覆盖率 |
|------|--------|
| 再动/额外行动 | 65% |
| 护卫/援护 | 58% |
| 驱散/净化 | 42% |
| 控制/反制 | 38% |
| 气力/能量积累 | 35% |

### 差异化机会
- 🎯 地形深度联动（仅15%覆盖率）
- 🎯 召唤物体系（仅12%覆盖率）
- 🎯 位置置换机制
- 🎯 预判/陷阱机制
